Task01：Ubuntu安装与配置

认识Ubuntu

  • 最流行的发行版
  • 与Rasbian同源于Debian
    • Raspberry Pi 的操作系统
  • 各发行版的指令系统基本一致
  • 版本获取：https://ubuntu.com/download/desktop

虚拟机

  • 物理机安装：先装win，再装Ubuntu
  • 虚拟机安装：最保险的方式
  • Linux Subsystem for Windows 10
  • 宿主中虚拟出软硬件
  • 相对的：客户机
  • 宿主机于客户机完全独立
  • 虚拟机软件
    • VMware：Win、Mac跨平台
    • Parallels Desktop：Mac
    • Hyper-V：内置在Windows Pro+，Server
    • Virtual Box：开源，Win、Mac、Linux跨平台
      • 拓展：比较虚拟机与Docker
  • 开始之前
    • 打开BIOS中的虚拟化选项
    • 不同主板BIOS选项各有差异
    • Virtualization Intel Virtual Technology VM
